# Options

Ce menu vous permet d’accéder rapidement à quelques paramètres fréquemment utilisés.

Préférences…  
Ouvre la boite de dialogue [\#chapter.dialogs.preferences](#chapter.dialogs.preferences).

Sur macOS, cet élément de menu se trouve dans le menu OmegaT.

Traduction automatique  
-   [\#dialogs.preferences.mt.automatically.fetch.translations](#dialogs.preferences.mt.automatically.fetch.translations)

Permet d’activer ou de désactiver les moteurs de traduction automatique disponibles.

Si plusieurs moteurs de traductions sont sélectionnés, vous pouvez aussi utiliser <span class="keycombo">C+M</span> pour passer de l’un à l’autre et insérer le dernier moteur sélectionné. Voir les préférences [\#dialogs.preferences.mt](#dialogs.preferences.mt) pour en savoir plus.

Glossaires  
-   [\#dialogs.preferences.dictionary.use.fuzzy.matching.for.dictionary.entries](#dialogs.preferences.dictionary.use.fuzzy.matching.for.dictionary.entries)

Permet d’activer ou de désactiver les correspondances partielles pour les entrées de glossaire.

Voir le paramètre [\#dialogs.preferences.glossary](#dialogs.preferences.glossary) pour en savoir plus.

Dictionnaires  
-   [\#dialogs.preferences.dictionary.use.fuzzy.matching.for.dictionary.entries](#dialogs.preferences.dictionary.use.fuzzy.matching.for.dictionary.entries)

Permet d’activer ou de désactiver les correspondances partielles pour les entrées de dictionnaire.

Voir le paramètre [\#dialogs.preferences.dictionary](#dialogs.preferences.dictionary) pour en savoir plus.

Saisie automatique  
Sélectionner les différentes options pour les activer ou les désactiver Voir le paramètre [\#dialog.preferences.auto.completion](#dialog.preferences.auto.completion) pour en savoir plus.

Filtres de fichiers généraux…  
Ouvre les préférences de [\#dialogs.preferences.file.filters](#dialogs.preferences.file.filters).

Utiliser [\#menus.project](#menus.project)[\#menus.project.properties](#menus.project.properties) pour accéder aux préférences de filtres de fichiers locales.

Règles de segmentation générales…  
Ouvre les préférences de [\#dialogs.preferences.segmentation.setup](#dialogs.preferences.segmentation.setup).

Utiliser [\#menus.project](#menus.project)[\#menus.project.properties](#menus.project.properties) pour accéder aux préférences de règles de segmentation locales.

Éditeur…  
Ouvre les préférences de [\#dialogs.preferences.editor](#dialogs.preferences.editor).

Accéder au dossier de configuration  
Ouvrir le dossier local où sont stockés les fichiers de configuration d’OmegaT.

L’emplacement dépend du système d’exploitation et des options de lancement. Voir l’annexe [\#configuration.folder](#configuration.folder) pour en savoir plus.
