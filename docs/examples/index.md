# Exemples et fixtures

Ce dossier contient des jeux de tests, des fixtures et des schémas utilisés pour valider les agents et les prompts.

## Contenu

- `agent-io/` — entrées/sorties attendues pour chaque agent.
- `prompts/` — versions texte des prompts prêts à l’emploi.
- `fixtures/` — chapitres source et traductions de test.
- `schemas/` — JSON Schemas des contrats agent.

## Utilisation

Les tests unitaires et d’intégration du projet NovelTrad 2.0 chargent ces fichiers comme données de référence.
